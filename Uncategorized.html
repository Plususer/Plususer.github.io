With this method, you can save your litle time :p. So here wo go.

Steps

  • Get the latest firmware for your device: Here
  • Download it then extract it.
  • Reboot to odin download mode.
  • Go to the folder where you extracted the firmware: Open Odin 3.09 Load firmware Select erase all option Then click on start:
  • Wait for 5 Minutes After that your device is reboot.
  • Congrats you'r successfully done :)

Here’s how to use the Recommended Settings for vegito kernel...

Use Kernel Adiutor to apply settings!

    For Balanced Performance

  • Set Min. to 200MHz and Max. to Max. Available for both Clusters.
  • Interactive/Impulse Governor for both Clusters.
  • Enable Schedule WorkQueues Toggle.

    • For Battery-Saving and Less Heat

  • Set Min. to 200MHz and Max. to 1200MHz for big Cluster.
  • Darkness/LionFish Governor for both Clusters.
  • Enable Schedule WorkQueues Toggle.

    • Thermal Control.

  • Enable Core Control. Temperature Throttle at 45 C.

    • GPU Profile.

  • Enable Adreno IDLER.

    • KCAL- Color Control

  • LiveDisplay ---Night Mode
  • Minimum RGB Value---32
  • Saturation Intensity---48

    • Wake Controls TAB

  • As per your own preference.

    • Sound TAB

  • As per your preference.

    • Note

  • In order to reset settings to default, just Disable the Apply On Boot option of the particular TAB in Kernel Adiutor and reboot the phone.
