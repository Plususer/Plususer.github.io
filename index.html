How To Install Vegito Kernel ?

    This simple guide will teach you How To Install Vegito Kernel on android devices which is supported by vegito.So what are you waiting for ? let's do this.

    Pre-Requirements

    • More than 60% battery on your device.
    • Make Sure that Your device has TWRP Installed (athene) Because You have to flash the kernel.zip from the TWRP menu.

      • Warning!

    • Do NOT flash this over other custom kernels!
    • You should only flash Vegito over the kernel that came with your ROM!!!
      • Download the Kernel file and transfer them to the internal memory or external memory of your device.
      • Then Boot your device into TWRP Recovery
      • Once you are in TWRP Recovery, create a backup of your current ROM from the TWRP Recovery main menu
      • Then Go to install Option (main menu TWRP) then select the Vegito kernel Zip file and swipe to flash
      • After the installation completes, go to the main menu of TWRP and reboot system

        • Setting up building environment.

        First, we install the required packages in our Ubuntu setup.. To do that, fire up a terminal window, and type : sudo apt-get install git-core gnupg flex bison gperf build-essential zip curl libc6-dev libncurses5-dev:i386 x11proto-core-dev libx11-dev:i386 libreadline6-dev:i386 libgl1-mesa-glx:i386 libgl1-mesa-dev g++-multilib mingw32 openjdk-6-jdk tofrodos python-markdown libxml2-utils xsltproc zlib1g-dev:i386 git

        Now, it's a good idea to start fresh.. So go ahead and reboot the system..

        Install Java

      • sudo add-apt-repository ppa:webupd8team/java
      • sudo apt-get update
      • sudo apt-get install openjdk-8-jdk
      • Done. Now reboot your system.
